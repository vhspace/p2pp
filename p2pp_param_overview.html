P2PP Parameter Overview

Name Description Default value
ABSOLUTEEXTRUDER Convert file to absolute extrusion FALSE
ACCESSORYMODE_MAF Generate MAF file (accessory mode) FALSE
ACCESSORYMODE_MAFX Generate MAFX file (accessory mode) FALSE
ACCESSORYMODE_MSF Generate MSF file (accessory mode) FALSE
AFTERSIDEWIPEGCODE GCode line(s) to be executed after sidewipe sequence. Multiple AFTERSIDEWIPECODE lines can occur. They are inserted in the sequence they appear in the config block Empty
AUTOADDPURGE Automatically add purge to meet minsplice requirements (only with fullpurge reduction and sidewipe) FALSE
AUTOLOADINGOFFSET Unused? 0
BEDORIGINX X-position of the origin of the bed, since 5.1.0 taken from file, can be overridden Input file
BEDORIGINY Y-position of the origin of the bed, since 5.1.0 taken from file, can be overridden Input file
BEDSIZEX X-size of the bed, since 5.1.0 taken from file, can be overridden Input file
BEDSIZEY Y-size of the bed, since 5.1.0 taken from file, can be overridden Input file
BEFORESIDEWIPEGCODE GCode line(s) to be executed before sidewipe sequence. Multiple BEFORESIDEWIPECODE lines can occur. They are inserted in the sequence they appear in the config block Empty
BIGBRAIN3D_BLOBSIZE See BB3D section in this documentation 40
BIGBRAIN3D_BLOBSPEED See BB3D section in this documentation 200
BIGBRAIN3D_COOLINGTIME See BB3D section in this documentation 12
BIGBRAIN3D_ENABLE See BB3D section in this documentation FALSE
BIGBRAIN3D_FAN_OFF_PAUSE See BB3D section in this documentation 0
BIGBRAIN3D_LEFT_SIDE See BB3D section in this documentation FALSE
BIGBRAIN3D_MOTORPOWER_HIGH See BB3D section in this documentation 450
BIGBRAIN3D_MOTORPOWER_NORMAL See BB3D section in this documentation 300
BIGBRAIN3D_NUMBER_OF_WHACKS See BB3D section in this documentation 1
BIGBRAIN3D_RETRACT See BB3D section in this documentation 3
BIGBRAIN3D_PRIME_BLOBS See BB3D section in this documentation 0
BIGBRAIN3D_PURGEPOSITION See BB3D section in this documentation 256.5
BIGBRAIN3D_PURGEPOSITIONY See BB3D section in this documentation Empty
CHECKVERSION Check online for new version FALSE
CONSOLEWAIT Wait for user input at end of processing even if there are no errors FALSE
DEBUGTCOMMAND Do not use FALSE
DO_NOT_GENERATE_M0 Prevents M0 commands in the gcode. M0 is being interpreted by the Palette device as a pause FALSE
EXTRAENDFILAMENT Amount of extra filament added at the end of the print in the last used color (in mm) 150
FINISH_MOVES_M400 Replaces G4 P0 command to M400. This can be mandatory for some printer firmwares in order to clear there movement buffers and sychronize their motion to the pings of the Palette device FALSE
FULLPURGEREDUCTION Engages full purge reduction FALSE
IGNOREWARNINGS Closes P2PP after processing even if there were warnings FALSE
KLIPPER_TOOLCHANGE Detect KLIPPER TOOLCHANGE commands instead of default Tn commands FALSE
LINEARPINGLENGTH Sets the length between pings programmed in the file 350
MANUAL_SWAP Pauses the print with M25 on toolchange. This is commonly not used/set FALSE
MINSPLICE Minimal splice length 70
MINSTARTSPLICE Minimal first splice length 100
P3_HOSTNAME Hostname used to connect to the Palette 3 (Pro) for file uploading Empty
P3_MINIMALTOTALFILAMENT Define the minimum amount of total filament consumed in mm. This is commonly not used/set 0
P3_PROFILENAME Change the name of the printer in your splice file. It has no functional benefit to change this parameter Prusa Slicer - P2PP Output
P3_PROCESSPREHEAT Allows the Palette 3 (Pro) to preheat the printer. This only works in coneccted mode FALSE
P3_ROTATEINTERFACE Rotate the web interface of the Palette 3 (Pro) by 180 degrees FALSE
P3_SHOWPRINTERPAGE Show a web interface if the Palette 3 (Pro) FALSE
P3_UPLOADFILE Trigger for uploading your processed file to the Palette 3 (Pro) FALSE
P+LOADINGOFFSET Accessory mode parameter for Palette+ Empty
P+PPM Accessory mode parameter for Palette+ Empty
PALETTE3 Define your Palette device as a Palette 3 FALSE
PALETTE3_PRO Define your Palette device as a Palette 3 Pro FALSE
POWERCHAOS Special feature request to allow sub 300 mm pings FALSE
PRINTERPROFILE Printer profile string length 16 for P2 and before 32 bit for P3 50325050494e464f
PURGESPEEDADJUST Multiplier for the purge speed ranging between 10% and 500% [0.1 - 5.0] 1.0
PURGETOPSPEED Set speed limit during purging. Value <200 is interpreted as mm/sec, larger values are interpreted as mm/min Empty
PURGETOWERDELTA Engages power delta FALSE
SAVEUNPROCESSED Save a copy of the unprocessed file to disk (useful for debugging) FALSE
SIDEWIPECORRECTION Correction factor applied to the filament extruded during side wipe 1
SIDEWIPELOC See sidewiping section Empty
SIDEWIPEMAXY See sidewiping section 175
SIDEWIPEMINY See sidewiping section 25
SPLICEOFFSET Defines the extra length in mm added to the first splice 30
TEMPERATURECONTROL Enables active temperature control by introducing controlled temperature waits during the print to allow for cooldown/heatup matching the print higher/lower temp requirements FALSE
WIPEFEEDRATE Defines the default feedrate used for wiping 3000
